# Jered website template

This website template is a re-designed from Jeren Free Website Template. It is a Dark on Light design developed by "OS Templates". The original work can be found at [OS templates](https://www.os-templates.com/free-website-templates/jeren) and the demo at [OS templates](https://www.os-templates.com/free-website-templates/jeren) website.
